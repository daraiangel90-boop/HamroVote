This project won the CivicCode2026 hackathon. HamroVote is a 3-man team project that works on solving the current problem of Nepal.
Recently, the prime minister of Nepal in an parliament session pledged the government to solve the issue of NRNs(Non Residential Nepalese) people to vote while being away from their motherland.
Thus, To solve that issue we have created a website that allows NRNs to vote which is free of data tempering through blockchain and avoid risk of influence through live camera detection or verification.
